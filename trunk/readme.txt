Adds a couple of lorem ipsum paragraphs to any blank page or post.


== Description ==

Don't waste time. Add some quick test text, with lorem ipsum.


== Installation ==

1. Upload 'add-some-text' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
